## 🎯 What I'm Learning

### Phase 1: Basics & Fundamentals
- Data types (string, number, boolean, null, undefined, symbol, bigint)
- Variables: `let` vs `const` vs `var`
- Functions (declarations, expressions, arrow functions)
- Conditionals (if/else, switch, ternary operators)
- Loops (for, while, do...while, for...of, for...in)
- Object literals & array basics

### Phase 2: Data Structures
- Arrays: methods, iteration, manipulation
- Objects: properties, methods, destructuring
- Maps & Sets
- Custom data structures: Stacks & Queues

### Phase 3: DOM Manipulation
- Selectors (`querySelector`, `getElementById`, etc.)
- Traversing the DOM
- Creating, removing, and modifying elements
- Event listeners & event delegation
- Show/hide functionality

### Phase 4: How JavaScript Works
- Execution context
- Call stack
- Event loop
- Hoisting & scope
- Closures

### Phase 5: Object-Oriented Programming
- Constructor functions
- Prototypes & prototype chain
- Inheritance
- ES6 Classes
- Protected properties
- Getters & Setters

### Phase 6: Asynchronous JavaScript
- AJAX & XHR
- Fetch API
- Callbacks & callback hell
- Promises (creation, chaining, error handling)
- Async/Await syntax

### Phase 7: Error Handling
- Try...Catch blocks
- Throwing custom errors
- Handling HTTP errors
- Finally blocks

### Phase 8: Web & Browser APIs
- Canvas API for drawing
- Geolocation API
- Audio/Video API
- Web Animations API
- Local & Session Storage

### Phase 9: Modules & Tooling
- CommonJS modules
- ES6 Modules (import/export)
- Webpack configuration
- Loaders & Plugins
- Babel for transpilation
- Modern development workflow

### Phase 10: Unit Testing
- Jest testing framework
- Writing test suites
- Testing algorithms:
  - Palindromes
  - Anagrams
  - FizzBuzz
  - Array chunking
  - Reverse strings

### Phase 11: Node.js & Express
- Node.js modules
- NPM package management
- Building REST APIs
- Express.js framework
- MongoDB integration
- Middleware & routing

### Phase 12: Full Stack Development
- Combining Express API with Webpack frontend
- No frontend frameworks - vanilla JS
- API design & frontend integration
- Deployment considerations
