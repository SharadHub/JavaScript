# Console Operations in JavaScript

This folder demonstrates various console methods and operations available in JavaScript for debugging and logging purposes.

## Topics Covered:
- Basic console logging with `console.log()`
- Error and warning messages using `console.error()` and `console.warn()`
- Displaying tabular data with `console.table()`
- Grouping console messages with `console.group()` and `console.groupEnd()`
- Styled console logs using CSS formatting

## Key Methods Demonstrated:
- `console.log()` - Standard output
- `console.error()` - Error messages
- `console.warn()` - Warning messages
- `console.table()` - Tabular data display
- `console.group()` / `console.groupEnd()` - Grouped output
- `%c` formatting for styled console output

## How to Use:
Open the `index.html` file in a browser and inspect the console to see the output of the JavaScript code.
