# Type Conversion in JavaScript

This folder demonstrates explicit type conversion in JavaScript, showing how to convert values from one data type to another.

## Topics Covered:
- Converting strings to numbers using `parseInt()` and `Number()`
- Converting numbers to strings using `.toString()` and `String()`
- Explicit vs. implicit type conversion
- When and why to use explicit type conversion

## Key Concepts:
- **Explicit Conversion**: Converting values manually using built-in functions
- `parseInt()` converts a string to an integer, ignoring non-numeric characters after numbers
- `Number()` converts a value to a number, returning NaN if conversion fails
- `.toString()` method converts a value to its string representation
- `String()` function converts any value to its string representation
- Type conversion is essential when dealing with user input or data from external sources

## How to Use:
Review the commented examples in `script.js` to see how to perform explicit type conversions in JavaScript. Uncomment the examples to test them in your environment.
