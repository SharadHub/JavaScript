# JavaScript Numbers

This project folder explores various built-in properties and methods of the `Number` object in JavaScript. It is designed to demonstrate basic number manipulation in JavaScript.

## Overview

The `app.js` file demonstrates how to use the following methods and properties:
- `toString()`: Convet a number into a string.
- `toFixed()`: Format a number with fixed decimal places.
- `toPrecision()`: Format a number to a specific total precision.
- `toExponential()`: Represent a number in exponential notation.
- `toLocaleString()`: Format a number using locale-specific sets of rules (e.g., Arabic-Egypt `ar-EG`).
- `valueOf()`: Retrieve the primitive value of a number object.
- `Number.MAX_VALUE`: Get the largest representable positive number in JavaScript.
- `Number.MIN_VALUE`: Get the smallest representable positive number in JavaScript.

## Usage

To run the examples and see the output, execute the following command in your terminal:

```bash
node app.js
```

Alternatively, open `index.html` in your browser and check the developer console to view the output.
